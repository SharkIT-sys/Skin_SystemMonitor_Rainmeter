## ✨ Features

- **CPU Monitoring**: Real-time CPU utilization.
- **Dual GPU Support**: Seamlessly monitors both **NVIDIA** and **Intel** integrated graphics.
- **RAM Usage**: Tracks physical memory usage with GB values and percentage.
- **Dual Storage Tracking**: Monitors free space on both C: and D: drives.
- **Network Traffic**: Displays live Download and Upload speeds.
- **Dynamic Color Alerts**: Colors automatically change based on resource usage:
  - 🔵 **Normal** (Under 75%)
  - 🟡 **Warning** (75% to 89%)
  - 🔴 **Critical** (90% and above)

## 🚀 Installation

### Prerequisites
1. Download and install [Rainmeter](https://www.rainmeter.net/) (version 4.5 or higher recommended).
2. For NVIDIA GPU monitoring to work, ensure you have the NVIDIA drivers installed (uses `nvidia-smi` natively).

### Setup
1. Clone this repository or download it as a ZIP file.
   ```bash
   git clone https://github.com/SharkIT-sys/Skin_SystemMonitor_Rainmeter.git
   ```
2. Copy the `Skin_SystemMonitor_Rainmeter` folder to your Rainmeter skins directory, usually located at:
   ```text
   C:\Users\YOUR_USERNAME\Documents\Rainmeter\Skins\
   ```
3. Open Rainmeter, click **Refresh all** at the bottom left.
4. Expand the `Skin_SystemMonitor_Rainmeter` folder, select `SystemMonitor.ini`, and click **Load**.

## 🎨 Customization

You can easily modify the colors, fonts, and sizes by editing the `@Resources/Variables.inc` file.
Just right-click the skin, select `Edit skin`, and adjust the variables to your liking!
